应急灯的光圈刚推开走廊三米的黑暗，苏晚晴的手就松开了我的衣服。

不是我挣开的。是她自己松开的——五根手指像失去了神经指令一样，一根接一根地从我冲锋衣的下摆上滑了下去，软塌塌地垂在她身侧。

“晚晴？”

她没有回答我。

她站住了。两条腿钉在走廊的防静电地砖上，膝盖微微内扣，整个人从上到下绷成了一张弓。她的呼吸还在——我能看到她米白色开衫下胸口的起伏——但她的眼睛已经不再看我了。

她在看走廊左侧，档案室的玻璃门。

应急灯的光扫过玻璃的时候，一块模糊的、暖黄色的光斑映在门后第三排档案柜的侧板上。柜子旁边站着一个轮廓。我第一眼的反应是真人——身高大概一米五出头，背有点驼，重心落在右脚上，一只手扶着柜沿，另一只手垂在身前，手指微微弯曲，像刚要伸手去取一本档案夹又被什么打断了。

然后那个轮廓转过了头。

不是转身。是转头。脖子以上缓慢地、一格一格地转动，像被锈住的齿轮在硬推。它的五官在玻璃后面始终模糊不清，但那个转过来的角度让我后颈的汗毛全部炸了起来——它在看苏晚晴。不是看我们，是看苏晚晴一个人。

“小晴。”

声音穿过玻璃，穿过应急灯的光，穿过我的肋骨，像有人把一根冰冷的指尖点在了我的胸骨上。

苏晚晴的嘴唇张开了。

她的嘴唇干裂，下唇正中一道细小的血口子渗着暗红色，但她的嘴角在上扬——不是被外力扯动的那种上扬，是从面部深层肌肉驱动出来的、带着完整情绪弧度的笑。

“奶奶？”她说。

她说话的语调往上飘，像一个小了二十岁的孩子。

档案室里的轮廓伸出了那只扶着柜沿的手。五指慢慢张开，手心朝上，手指微微勾起——那不是招呼，是邀请。一个这辈子重复过几千次、已经刻进肌肉记忆的动作：伸手，等一个孩子跑过来牵。

我转身一步跨到苏晚晴面前，双手抓住她的肩膀，把她拧过来面对我。她的脸被应急灯从侧面打着——她瞳仁里映着的不是我的倒影，是档案室玻璃门上的光斑。她的瞳孔放大到虹膜的边缘只剩一个细环，嘴角仍挂着那个不属于三十二岁成年人的笑容。

“那不是你奶奶，”我压低声音说，“苏晚晴，你看着我——那不是你奶奶。”

她听见了。她的睫毛颤了两下，视线在我脸上扫过去，像风吹过一页没压住的书页，然后又被翻回了档案室那边。

她从我手指间滑了出去。

不是推开的。是整个人像被一根无形的绳子拽着，肩膀往前送，脚步带着身子从我双手中脱了出去。她的开衫料子从我指尖擦过去，米白色的绵软触感被抽走，取而代之的是冰冷的、带着灰尘味的空气。

“小晴，奶奶在这儿。过来。”

那个声音又响起来了。这一次隔着玻璃听起来更清晰了，能分辨出具体的音色——那位老妇人的声带已经松弛了，说话带着气音，尾音微微发颤，像冬天窗台上被风吹动的旧报纸。她用的不是普通话，是苏州话，那种软糯的、把“你”说成“倷”的吴侬软语。

苏晚晴朝档案室的门迈出了第四步。

第五步。

我追上去。左臂从她背后绕过去箍住她的腰，右手把自己的冲锋衣从肩上扯下来，翻了个面，兜头把她的脸、她的耳朵、她头顶散落的碎发全部裹了进去。她撞进我怀里的时候身体是僵直的——不是恐惧的僵硬，是那种被催眠后对外力完全没有反应的僵硬。

我收紧手臂，把她整个上半身锁在我胸口。隔着冲锋衣的尼龙布，我能感觉到她的头在左右转动，不是挣扎，是在找声音的方向。她呼出的热气透过布料闷在我锁骨上，急促，紊乱，呼气的频率比吸气快三倍。

“奶奶给你煮了酒酿圆子，就摆在厨房台子上。”那个声音继续说，语气温柔得像在哄一个刚放学回家的小学生，“倷放学转来先吃，冷脱仔就弗好吃哉。”

苏晚晴的身体在我怀里弓了起来。

她的膝盖突然往下坠，像被抽掉了关节里的螺栓。我的重心被她带着往前倾，左脚往前跨了半步才勉强撑住。我半蹲下去，把膝盖垫在她背后，让她躺在我大腿上，一只手隔着外套按住她的后脑勺压在我胸口，另一只手去摸裤兜。

打火机。

我的拇指拨了一下砂轮——火星溅开，没点着。又拨一下，火苗窜起来，幽蓝色的焰心在走廊空调残存的冷风里抖得像一片将落的叶子。我捏着火机往前探，再用牙齿咬住酒精喷雾的瓶盖把它拽开，把喷嘴对准档案室的方向连续扣动了三下。

细密的水雾喷出去，在半空中被应急灯的光照成一片短暂的白虹。

我把打火机的火焰凑上去。

橘红色的火墙在走廊里炸开。酒精燃烧的热浪扑面而来，我眯起眼睛，感觉到眉毛前端的毛发被热气烤得卷了一下。火舌从地面窜到天花板，把整条走廊照得像白昼——玻璃门里的档案柜、倒在地上的转椅、天花板上消防喷淋头的金属截面，全部被映成了灼热的橘色。

那个轮廓在火焰中崩塌了。

不是躲开，不是退后。是边缘先开始溶解——棉袄的肩线塌了下去，驼背的弧度像被热水浇过的糖画一样化开，然后整条伸出的手臂从手指到肘部逐一溃散成黑雾。黑雾在玻璃后面翻涌了一下，然后被不知哪来的、似乎比黑暗更深的黑暗吞没。

最后消失的是那个微笑的感觉。

火舌在三秒内退尽。空气里弥漫着酒精燃烧后的焦味，呼吸进肺里是热的。

我低头去看苏晚晴。她缩在我的冲锋衣下面，整个人蜷成小小的一团——一米六不到的身高，此刻团起来的体积可能还不如一个高中生塞满的书包。她的手指不知道什么时候从开衫袖口里伸了出来，死死攥着我的T恤下摆，指甲隔着棉布掐进我腰侧的肉里。

疼。但我没说。

我把外套从她头上掀开一角。应急灯的暖光落在她脸上——妆已经花了，眼线化开的黑灰色从下眼睑一直晕到颧骨上方，脸上那道士力架蹭的灰印还在，嘴唇干裂处渗出的血珠已经凝固成一小块暗红色。她的眼睛睁着，瞳孔缩成了针眼大的两个黑点，泪水的膜覆在眼球表面，把虹膜的颜色泡得模糊不清。

但她没有哭。

她从一个被几千万人盯着长大的全民偶像，此刻本能反应仍然是憋着。

我伸手把粘在她脸颊上的一缕头发拨开。指尖碰到她的皮肤——冰凉的，但不是死人那种凉，是冬天在外面走久了之后露在外面的皮肤温度。血液还在流动的证据。

“苏晚晴。”

她的视线慢慢聚焦到我脸上。

“你这表情比我自己画的恐怖游戏宣传图还劝退，”我低头看着她，“我那个游戏在Steam上被喷得最狠的就是美术，你现在这张脸可以直接拿去当精神续作的封面——‘买了就做噩梦，差评如潮’。”

她眨了眨眼。

泪膜破了，两行水分沿着她的脸侧滚落下去，但她的嘴角也同时被扯了一下——往上。不是笑，是被笑和哭同时拉扯之后产生的、不受控的肌肉痉挛。

“你——”她嗓子哑得只剩下气音，“你脑子里到底装的什么。”

“黄色废料和游戏攻略，外加百分之四十的碳水。”我说，“恐怖片定律——话越多活得越久，沉默寡言的帅逼第一个领便当。”

她发出了一声短促的、像是在笑又像是在抽泣的气音。然后她整个人就散了——不是昏过去，是那个从十六岁就被经纪公司、媒体和粉丝合力焊上的架子，终于出现了第一道看得见的裂痕。

她的肩膀塌下去，头在我胸口拱了两下，然后攥着我T恤的手指松了力，开始哭。不是嚎啕，是那种压抑了十几年之后终于被捅破一个口的哭法——每一口气吸进去都在发抖，呼出来的时候带着断断续续的、堵在喉咙口的呜咽。她的眼泪浸透了我T恤的棉布，温热的液体顺着我的胸口往下淌。

我靠着档案室门外的墙滑坐下去，把腿伸直在走廊的地砖上，把她整个人兜在怀里。应急灯放在我们旁边的地上，光圈罩着她的背和我的膝盖。

“那东西拿你的记忆做诱饵，”我低声说，声音放得很轻，像是在跟她脊背上还在发抖的肌肉对话，“它选了你最想见的人，最想听的话。这叫精准精神攻击，跟我在游戏里设计的那套SAN值打击逻辑一模一样。”

她的哭声顿了一下，换了口气。没说话，但头在我胸口微微转了一个角度——她在听。

“我给它起个名字吧，叫‘拟态阴影’。行为模式类似于RPG里的魅惑怪，读取目标记忆里最高情感权重的人物形象，然后生成幻象引诱目标主动接近。目测有效范围至少十米，隔着玻璃也能施放。弱点是怕火怕光，刚才酒精喷雾一烧就退了。”

她在我怀里哼了一声。又闷又哑的。

“但有个问题我没想通。”我说。

她吸了一下鼻子。“什么？”

“它读取记忆是读什么格式的？比如你奶奶的声音和说话习惯，它模仿得很精准，但我妈的声音它模仿得不对——它叫我全名，但我妈只叫我小名。要么是数据库不完整，要么是它读取记忆的时候有精度限制，只能抓到大概率相关的数据。”

苏晚晴慢慢抬起了头。

她的眼睛肿了，眼白里全是血丝，瞳孔已经恢复了正常的大小。应急灯的光在她眼眶里映出两个微小的亮点，很亮，像从很深的水井底部看到的星光。

“你妈叫你什么？”她问。声音还是哑的，但语调已经稳了。

“林子。”

“因为这个你才没被骗？”

“因为这个我刚才在走廊里犹豫的时间不超过零点五秒。”我低头看她，“如果它叫的是‘林子’，我可能在那一瞬间就走出去了。”

她看着我的脸，沉默了片刻。

然后她的一只手从我T恤上松开，往上移了大概十公分，搭在我左臂的肘弯处。掌心的温度很凉，但手指扣上来的时候用的力道是握，不是抓——是她主动选择握住的，不是被恐惧捏合的反射。

“你妈也是生病？”她问。

“宫颈癌。五年前的事。我当时在上海做游戏宣发，签合同的时候接到电话，飞机赶回去的时候人已经走了。”我看着应急灯的灯罩，“没见到最后一面。跟你一样。”

她的手指在我肘弯上收紧了一下。

我们就这样靠着档案室的墙坐了一会儿。应急灯的光圈外黑暗沉寂无声，酒精的焦味已经散得差不多了，空气重新变回了中央空调残存的那种低温的、微微带着灰尘腥气的味道。档案室的玻璃门后什么也没有，只有第三排档案柜安静地站在暗处。

然后苏晚晴的肚子叫了一声。

不是隐喻，是真真切切的、从胃部挤上来的咕噜声。在安静得像太平间的走廊里，这一声清楚得像是往池子里丢了颗石子。

苏晚晴低下头看了看自己的肚子，又抬头看我。她的表情是哭过之后特有的那种空白的平静，眼睛下面挂着泪痕，但嘴角抿成一条横线，像是在说“对，就是我，怎么了”。

“我饿了。”她说。

“看到了。准确说是听到了。”

“我俩加起来今天晚上消耗的热量大概够跑一场半马，”她把额前挡眼的头发往后捋了捋，声音恢复到了能正常断句的水平，“而且刚才你把最后一点体力也耗在抱我身上了。”

“你这句话有歧义。”

她愣了一下，反应过来，白了我一眼。红肿着眼眶白我一眼，杀伤力接近于零。

“走吧。”我从地上撑起来，把冲锋衣重新穿上，拉链拉到胸口，“便利店在休息区方向，刚才看到指示牌——‘员工休息区→’，便利店大概率在那边。我们沿墙走，应急灯照前方，我走左边你走右边贴着我的右半身。”

苏晚晴站起来的时候腿还在打颤，但她没让我扶。她一只手撑着墙稳了五秒，等膝盖不抖了才松开，然后弯腰把掉在地上的酒精喷雾捡起来递给我。

“还剩大概五次喷射量，”我晃了晃瓶子，“打火机漏气了，还能点七八下。便利店里一般有打火机卖——顺便找找有没有手电筒，运动饮料，含糖分的那种。你现在需要补糖。”

她点头，然后很自然地往我右侧靠过来。不是紧紧粘着，是那种两个人共享体积空间的站位——她右肩跟我右臂之间的距离不超过三厘米，她的左手手指勾住了我冲锋衣右边口袋的边沿。不紧，只是勾着。

我们沿走廊贴墙前进。应急灯的光在前面铺开一个暖黄色的半圆，光圈边缘贴着墙根扫过去，掠过了紧闭的办公室门、翻倒的盆栽、一台屏幕碎裂但还挂在墙上的液晶广告屏。苏晚晴的脚步明显比出来的时候慢了，每一次抬脚都在压着呼吸——她的体力在透支的边缘，但她没吭声。这种不吭声比喊累更让我觉得麻烦。

“你的电量显示多少？”我说。

“嗯？”

“我说你的体力。手机快没电的时候会跳百分之二十的警告，你现在大概在百分之十五——还能走，但随时可能强制关机。”

她低低地笑了一声。“你这比喻对我没用，我手机的电池健康度常年百分之九十七以上。经纪人天天盯着我充电。”

“经纪人是王哥对吧？”

“嗯。”

“那个假王哥的声音你刚才在走廊里听到了，”我说，“现在你觉得它是提前录好的还是实时生成的？”

苏晚晴的脚步顿了一拍。然后继续走。

“实时生成的。”她说完停了停，“因为它叫我的语气是当下的——它知道我害怕，所以那个声音里才有那种安慰的语气。”

“精确。”我说，“所以这个怪物不只会读记忆，它还能读你的实时情绪状态。它知道你什么时候最脆弱，什么时候最容易动摇，然后在那个时刻打出最要命的牌。”

她沉默了一个转弯的距离。

“它还会再来吗？”

“会。”我把应急灯换到左手，右手摸到裤兜里的打火机，“但它每次进攻都消耗了能量——第一次在走廊里模仿我跟我妈，被酒精火烧退；第二次针对你一个人，又是特定记忆又是苏州方言，精度更高但也被打断。这两次它都没得手。它在学习，但也在暴露自己的限制。”

“什么限制？”

“它怕火怕光的弱点没变。两次都是火焰逼退，不是它自己撤退的。说明这个弱点是硬性的、物理层面的，不是策略性的示弱。”

走廊到了尽头。左转一个弯，再走十米，应急灯的光扫到了一块指示牌——蓝底白字，写着“员工休息区→”，下面一行更小的字：“自动贩卖区·24小时便利店”。

苏晚晴的步频明显快了半拍。

休息区是一个开阔的中庭空间。应急灯的光扫过去，能照到L型的布艺沙发组合、两张茶几、一排半死不活的盆栽绿植，以及靠墙一字排开的自动贩卖机。贩卖机的屏幕全黑，但玻璃面板上映着应急灯的暖光，能隐约看到里面整齐排列的饮料罐。最里面隔了一面落地玻璃墙，墙上贴着大幅的手写海报：“24小时便利店——扫码享会员价”。

玻璃墙后面，货架整齐地站在黑暗中。

苏晚晴走到饮料贩卖机前面，伸手去摸玻璃，然后又缩了回来。“常温。”她说。

“常温也行。”我拉开贩卖机的取物口挡板——没电，磁吸锁失效，挡板轻轻一拉就开了。我从里面拎出两瓶矿泉水、一瓶蓝色运动饮料，全塞进苏晚晴怀里。“先补液，少量多次。”

便利店是推拉玻璃门，没锁，轻轻一拉就开了。应急灯照进去，货架上的膨化食品、泡面桶、独立包装的面包和常温牛奶排列得整整齐齐，像是在等着最后一批顾客光临。空气里飘着一股塑料包装和隔夜关东煮汤底混合的复杂味道——不好闻，但在这种境况下，食物的存在本身就是一种信号。

安全区内。

苏晚晴直接走到面包货架前，取下一袋手撕面包撕开咬了一口。她嚼了两下，然后停住了，表情微妙地转过头看我。

“潮的。”她说，“保质期还有三天，面包是潮的。”

“断电后冷藏系统停转，温差一大密封袋内凝水了。”我走过去从她手里掰了一块塞嘴里，“热量还在。客观的热量不会背叛你。”

她咽下去，又掰了一块继续吃。

我把应急灯放在收银台上，绕到柜台后面去搜刮。收银机旁边的塑料展示架上挂着五只一次性打火机——透明塑料壳，液面高度清晰可见，满的。旁边挂钩上还有一个小手电筒，七号电池的，塑料外壳，推开关的时候发出一声廉价的咔嗒声——亮了，白光，亮度不高，但比没有强得多。

“物资刷新。”我把打火机揣进冲锋衣口袋，“火源备件充足，还多一个手电。”

苏晚晴在收银台前吃第二块压缩饼干，嘴里含糊地应了一声，目光漫无目的地扫着四周。

然后她顿住了。

“林逸，”她指着收银台后面的墙面，“你看那个。”

我把应急灯举高，暖黄光扫上去——墙上贴着一张塑封的楼层平面图，蓝色底，白色细线，标注着各区域的名称、走道方向和消防通道的位置。图不大，信息很密。我们目前的位置在图的右下方——102层南翼办公区，便利店，休息区，档案室，厕所，全部对得上。

但让我后脑勺发麻的是图的下半部分。

那里有一张竖剖面的缩略图，标注了“天际方舟”从1层到300层的功能分区。在101层到120层之间，用一条加粗的红色虚线圈出来的带状区域，旁边印着一行红色小字：

“纵向防火隔断层——医疗站·避难大厅·应急仓库·独立供电系统”。

苏晚晴也看到了。

她的手指点在那四个字上——避难大厅。

“独立供电，”她读得很慢，每个字都像是放在舌头上掂了掂，“也就是说那里有灯。”

“有灯，有医疗物资，可能有通讯设备。而且‘纵向防火隔断层’意味着那几层楼的墙体用了更厚的防火材料，可能比普通楼层更难被侵入。如果这楼里还能有安全的地方，那里排第一。”

她看了我一眼，然后从收银台上拿起小半袋没吃完的压缩饼干重新塞进嘴里。咀嚼的速度比刚才快了三分之一。她的眼白还是红的，泪痕还没干透，但眼睛里面那个凝光的小点——应急灯在她瞳孔中心映出的那个微小的亮点——此刻是稳的。
